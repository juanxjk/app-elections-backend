# Setup

## Requirements

- Node.js v12+
- NPM v6+

## How-to

- Build
  ```
  npm run build
  ```
  - For Yarn users:
    ```
    yarn run build
    ```
- Run (_production mode_)
  ```
  npm run start
  ```
  - For Yarn users:
    ```
    yarn run start
    ```
- Run (_development mode_)
  ```
  npm run dev
  ```
  - For Yarn users:
    ```
    yarn run dev
    ```
